ūüß† How It Works
Nemesis is built around an asynchronous pipeline:

Crawler ‚Äď Starts from the target, respects scope/robots, and recursively follows links. It extracts forms, JavaScript endpoints, and parses sitemaps.

Fingerprinter ‚Äď Analyses headers and HTML to identify frameworks, CDNs, and WAFs.

Security Analyser ‚Äď Checks all relevant response headers, performs CORS misconfiguration tests, and runs a TLS scan (certificate, cipher, key size).

Vulnerability Scanner ‚Äď Injects test payloads into query parameters. Findings are only reported when a verification step succeeds (e.g., token reflected for XSS, SQL error message present, or redirect to external domain for open redirect). Sensitive files are flagged only when HTTP 200 is returned.

Reporter ‚Äď Collects all findings and statistics, renders them into a self‚ÄĎcontained HTML page, or exports as JSON/CSV.
